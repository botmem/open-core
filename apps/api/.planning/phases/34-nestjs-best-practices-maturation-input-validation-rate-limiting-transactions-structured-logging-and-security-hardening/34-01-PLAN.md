    2. Create DTO classes in `dto/` subdirectories within each module. One class per file. Use class-validator decorators:

    **user-auth/dto/register.dto.ts:**
    - `email: string` — @IsEmail(), @Transform lowercase+trim
    - `password: string` — @IsString(), @MinLength(8), @MaxLength(100)
    - `name: string` — @IsString(), @MinLength(1), @MaxLength(100), @Transform trim

    **user-auth/dto/login.dto.ts:**
    - `email: string` — @IsEmail(), @Transform lowercase+trim
    - `password: string` — @IsString(), @MinLength(1)

    **user-auth/dto/forgot-password.dto.ts:**
    - `email: string` — @IsEmail(), @Transform lowercase+trim

    **user-auth/dto/reset-password.dto.ts:**
    - `token: string` — @IsString(), @IsNotEmpty()
    - `newPassword: string` — @IsString(), @MinLength(8), @MaxLength(100)

    **accounts/dto/create-account.dto.ts:**
    - `connectorType: string` — @IsString(), @IsNotEmpty()
    - `identifier: string` — @IsString(), @IsNotEmpty()

    **accounts/dto/update-account.dto.ts:**
    - `schedule?: SyncSchedule` — @IsOptional(), @IsString(), @IsIn(['manual', 'hourly', 'daily', 'weekly'])

    **contacts/dto/update-contact.dto.ts:**
    - `displayName?: string` — @IsOptional(), @IsString()
    - `avatars?: Array<{ url: string; source: string }>` — @IsOptional(), @IsArray(), @ValidateNested({ each: true }), @Type(() => AvatarDto)
    - `metadata?: Record<string, unknown>` — @IsOptional(), @IsObject()
    - Create nested `AvatarDto` with @IsString() url and @IsString() source

    **contacts/dto/split-contact.dto.ts:**
    - `identifierIds: string[]` — @IsArray(), @IsString({ each: true }), @ArrayMinSize(1)

    **contacts/dto/merge-contact.dto.ts:**
    - `sourceId: string` — @IsString(), @IsNotEmpty()

    **contacts/dto/search-contacts.dto.ts:**
    - `query: string` — @IsString(), @IsNotEmpty()

    **contacts/dto/dismiss-suggestion.dto.ts:**

    **agent/dto/ask.dto.ts:**

    - `limit?: number` — @IsOptional(), @IsInt(), @Min(1), @Max(100)
    - Nested `AskFiltersDto`: sourceType?, connectorType?, contactId? all @IsOptional() @IsString()

    **agent/dto/remember.dto.ts:**
