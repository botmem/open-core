    **config.service.ts -- Production secret validation:**
    Add a `validateProductionSecrets()` method to ConfigService:
    ```typescript
    validateProductionSecrets(): void {
      if (process.env.NODE_ENV !== 'production') return;

      const defaults = [
        { name: 'APP_SECRET', value: this.appSecret, default: 'dev-app-secret-change-in-production' },
        { name: 'JWT_ACCESS_SECRET', value: this.jwtAccessSecret, default: 'dev-access-secret-change-in-production' },
        { name: 'JWT_REFRESH_SECRET', value: this.jwtRefreshSecret, default: 'dev-refresh-secret-change-in-production' },
      ];

      for (const { name, value, default: def } of defaults) {
        if (value === def) {
          throw new Error(`FATAL: ${name} is using default value in production. Set a secure value.`);
        }
      }
    }
    ```

    Also add `implements OnModuleInit` to the class and an `onModuleInit()` that calls `this.validateProductionSecrets()`. Import `OnModuleInit` from `@nestjs/common`.

    **agent.controller.ts -- Replace error swallowing with HTTP exceptions:**

    Remove the `fail()` helper function entirely. For each endpoint (`ask`, `timeline`, `remember`, `forget`, `context`, `summarize`, `status`):

    1. Remove the outer `try/catch` block that returns `fail(err.message)` -- let NestJS global exception filter handle unexpected errors (it already sends to PostHog via PostHogExceptionFilter).
    2. Remove manual input validation (e.g., `if (!body.query?.trim()) return fail('query is required')`) -- the DTOs from Plan 01 now handle this via ValidationPipe.
    3. Keep the `ok()` helper (it's fine for success responses).
    4. For `forget()`, if memory not found, throw `new NotFoundException('Memory not found')` instead of `return fail('memory not found')`.
    5. For `context()`, if contact not found, throw `new NotFoundException('Contact not found')`.

    Example final pattern for `ask()`:
    ```typescript
    @RequiresJwt()
    @Post('ask')
    @HttpCode(200)
    @Throttle({ default: { limit: 20, ttl: 60000 } })
    async ask(@Body() dto: AskDto) {
      const start = Date.now();
      const result = await this.agentService.ask(dto.query, {
        filters: dto.filters,
        limit: dto.limit,
      });
      const sources = [...new Set(result.results.map((r) => r.connectorType))];
      return ok(result, {
        queryTime: Date.now() - start,
        resultCount: result.results.length,
        sources,
      });
    }
    ```

    Import `NotFoundException`, `BadRequestException`, `InternalServerErrorException` from `@nestjs/common`.
