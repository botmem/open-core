    - forgotPassword: invalidates any existing unused reset tokens for the user before creating new one
    - resetPassword(token, newPassword): validates token hash lookup, checks not expired, checks not used, updates password, marks token used, revokes all refresh tokens
    - resetPassword: rejects expired token with 400
    - resetPassword: rejects already-used token with 400
    - resetPassword: rejects invalid token with 400

    1. Add to `users.service.ts`:
       - `updatePasswordHash(userId, newHash)` -- updates passwordHash and updatedAt
       - `createPasswordReset(userId, tokenHash, expiresAt)` -- inserts into passwordResets
       - `invalidateUserResets(userId)` -- sets usedAt on all unused tokens for user
       - `findPasswordReset(tokenHash)` -- returns reset row or null

    2. Add to `user-auth.service.ts`:
       - Inject MailService
       - `async forgotPassword(email: string)`:
         - Find user by email. If not found, return silently (no enumeration)
         - Generate token: `crypto.randomBytes(32).toString('base64url')`
         - Hash: `crypto.createHash('sha256').update(token).digest('hex')`
         - Invalidate existing unused resets for this user
         - Store hash with expiresAt = now + 1 hour
         - Build resetUrl: `${config.frontendUrl}/reset-password?token=${token}`
         - Call mailService.sendResetEmail(user.email, resetUrl)
       - `async resetPassword(token: string, newPassword: string)`:
         - Validate newPassword >= 8 chars
         - Hash submitted token with SHA-256
         - Look up in DB
         - If not found: throw BadRequestException('Invalid or expired reset token')
         - If usedAt set: throw BadRequestException('Reset token already used')
         - If expiresAt < now: throw BadRequestException('Reset token expired')
         - Hash new password with bcrypt(12)
         - Update user's passwordHash
         - Mark token as used (set usedAt)
         - Revoke all user's refresh tokens

    3. Add controller endpoints:
       - POST /forgot-password: body { email }, calls service.forgotPassword, returns { ok: true }
       - POST /reset-password: body { token, newPassword }, calls service.resetPassword, returns { ok: true }

       - `logout`: POST /api/user-auth/logout with credentials: 'include', clear state
       - `refreshSession`: POST /api/user-auth/refresh with credentials: 'include'. On success: update accessToken + user. On failure: clear state, return false
