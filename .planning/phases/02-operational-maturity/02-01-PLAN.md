<objective>
Implement a nightly decay job that recomputes recency-based weights for all memories via BullMQ, ensuring old unpinned memories naturally rank lower over time while pinned memories retain their scores.

Purpose: Without decay, the `weights` JSON column preserves stale recency scores from ingest time, meaning months-old memories rank as if they were recent. The decay job refreshes these weights nightly.

Output: `DecayProcessor` on a `maintenance` queue, scheduled at 3:00 AM by default via `upsertJobScheduler()`, processing memories in batches of 500.
</objective>
