This creates `.husky/` directory and adds `"prepare": "husky"` to root package.json scripts.

3. Replace the contents of `.husky/pre-commit` with:
```bash

```

4. Create `.husky/pre-push` with:
```bash
#!/usr/bin/env sh

# Run typecheck and tests on packages changed since origin/main
# Falls back to all packages if origin/main doesn't exist
if git rev-parse --verify origin/main >/dev/null 2>&1; then
  pnpm turbo typecheck test --filter='...[origin/main...HEAD]'
else
  pnpm turbo typecheck test
fi
```

5. Add lint-staged config to root `package.json`:
```json
{
  "lint-staged": {
    "*.{ts,tsx}": [
      "eslint --fix",
      "prettier --write"
    ],
    "*.{json,md,yaml,yml}": [
      "prettier --write"
    ]
  }
}
```

6. Verify hooks are installed: `.git/hooks/pre-commit` should exist and reference Husky.
