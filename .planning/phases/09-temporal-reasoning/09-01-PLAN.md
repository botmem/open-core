    3. GREEN: Create `apps/api/src/memory/nlq-parser.ts` implementing:

    **`parseNlq(query: string, refDate?: Date): NlqParsed`** -- main entry point:
    - Calls `parseTemporal(query, refDate)` for temporal extraction
    - Calls `classifyIntent(query)` for intent
    - Calls `detectSourceType(query)` for source type hint
    - Strips temporal tokens from query to produce `cleanQuery`
    - After stripping temporal text, also strips dangling prepositions (in, from, during, between, since, before, after, on) that immediately precede the temporal match position
    - If cleanQuery is empty after stripping, falls back to original query

    **`parseTemporal(query: string, refDate: Date): TemporalResult | null`**:
    - Uses `chrono.parse(query, refDate)` (NOT `parseDate`) to get `ParsedResult[]`
    - Takes first result only
    - Confidence check: require `result.start.isCertain('month') || result.start.isCertain('day')` -- reject low-confidence parses
    - Do NOT set `forwardDate: true` (chrono-node default resolves "January" to most recent past)
    - For single-point results (no `result.end`): if only month certain (not day), expand to full month; otherwise expand to end of day
    - Returns `{ range: { from, to }, text: result.text }` or null

    **`classifyIntent(query: string): 'recall' | 'browse' | 'find'`**:
    - Ordered regex patterns, first match wins:
      - Find: `/\b\w+'s\s+(phone|email|number|address|birthday)/i`, `/^find\b/i`, `/\b(phone|email|number|address)\s+(of|for)\b/i`
      - Browse: `/^(show|list|browse)\b/i`, `/\brecent\b/i`, `/^(my|all)\s+(photos?|emails?|messages?)/i`
      - Recall: `/^(what|who|when|where|how|why)\s+(did|was|were|is|said)/i`, `/\btell\s+me\s+(about|what)/i`
    - Default: 'recall'

    **`detectSourceType(query: string): string | null`**:
    - Pattern match: photos/photo -> "photo", emails/email -> "email", messages/message -> "message"
    - Returns null if no match

    **`NlqParsed` interface** (exported):
    ```typescript
    export interface NlqParsed {
      temporal: { from: string; to: string } | null;
      temporalText: string | null;
      intent: 'recall' | 'browse' | 'find';
      cleanQuery: string;
      sourceTypeHint: string | null;
    }
    ```
