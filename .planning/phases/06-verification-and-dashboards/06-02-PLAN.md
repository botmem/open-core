  <action>
Create saved dashboards and insights in PostHog EU (https://eu.i.posthog.com). These are all PostHog UI tasks -- no code changes needed.

**Step 1: Verify Web Analytics (WEB-01)**
Go to PostHog -> Web Analytics tab. This is a built-in PostHog feature that should already show:
- Page views count
- Unique visitors count
- Session counts
- Top pages table
If the Web Analytics tab is empty, ensure the PostHog project has web analytics enabled in Project Settings.

**Step 2: Create "Botmem Usage" Dashboard (PROD-01)**
Go to PostHog -> Dashboards -> New Dashboard. Name it "Botmem Usage". Add these saved insights:

- **Searches per day:** New Insight -> Trends -> Event: `memory_search` -> Display: line chart -> Date range: Last 30 days -> Save to dashboard
- **Syncs per day:** New Insight -> Trends -> Event: `sync_complete` -> Display: line chart -> Date range: Last 30 days -> Save to dashboard
- **Memories created:** New Insight -> Trends -> Event: `memory_created` (or check if the event name is `embed_complete` from Phase 2 analytics) -> Display: line chart -> Date range: Last 30 days -> Save to dashboard

**Step 3: Create Connector Setup Funnel (PROD-02)**
On the same "Botmem Usage" dashboard, add a new Funnel insight:
- Step 1: `$pageview` where `$current_url` contains `/connectors`
- Step 2: `connector_setup` event (this was added in Phase 4)
- Step 3: `sync_complete` event
- Funnel type: Conversion funnel
- Save to "Botmem Usage" dashboard

**Step 4: Create Search Retention Insight (PROD-03)**
On the same dashboard, add a new Retention insight:
- Cohort event: `memory_search` (first search)
- Return event: `memory_search` (subsequent searches)
- Period: Day
- Save to "Botmem Usage" dashboard
