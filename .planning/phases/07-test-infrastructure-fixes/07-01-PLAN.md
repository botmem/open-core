# Plan 07-01: Install coverage tooling and configure workspace scripts

## Goal
Install `@vitest/coverage-v8` across all workspace packages, standardize coverage thresholds to 80/80/80/75, and add a `pnpm test:coverage` root script that runs coverage for all packages with text + lcov reporters.

## Context
- 9 vitest configs exist across the monorepo (api, web, shared, connector-sdk, 5 connectors)
- `packages/cli` has NO vitest config — needs one created
- `@vitest/coverage-v8` is NOT installed anywhere
- Existing thresholds are inconsistent: api=80/75/80/80, slack=75/70/75/75, shared=90/85/90/90
- **CONSTRAINT: No functionality changes** — only tooling and config

## Tasks

<task id="1">
<files>package.json, .gitignore</files>
<action>
Install `@vitest/coverage-v8` as a root devDependency:
```bash
pnpm add -D -w @vitest/coverage-v8
```
Add `coverage/` and `*.lcov` to `.gitignore` if not already present.
</action>
<verify>
```bash
grep "coverage-v8" package.json && grep "coverage" .gitignore
```
</verify>
<done>@vitest/coverage-v8 in root devDependencies; coverage/ in .gitignore</done>
</task>

<action>
Update ALL 9 existing vitest configs to have standardized coverage settings. Create `packages/cli/vitest.config.ts` (new file). Add `"test": "vitest run"` to `packages/cli/package.json` if missing.

Target coverage block for all 10 configs:
```typescript
coverage: {
  provider: 'v8',
  reporter: ['text', 'lcov'],
  reportsDirectory: './coverage',
  thresholds: {
    statements: 80,
    branches: 75,
    functions: 80,
    lines: 80,
  },
},
```

Keep all existing config (plugins, environment, setupFiles, aliases) — only modify the `coverage` block.
</action>
<verify>
```bash
for f in apps/api/vitest.config.ts apps/web/vitest.config.ts packages/shared/vitest.config.ts packages/connector-sdk/vitest.config.ts packages/connectors/*/vitest.config.ts packages/cli/vitest.config.ts; do echo "=== $f ===" && grep -A4 "thresholds" "$f"; done
```
</verify>
<done>All 10 vitest configs have identical thresholds (80/80/80/75) and reporter: ['text', 'lcov']</done>
</task>
